# PlanMeasure — Area Takeoff Tool
A free, open-source web tool for calculating areas from floor plan drawings. Built as a single self-contained HTML file — no backend, no dependencies to install, no data leaves your browser.
Features

Upload PDF (multi-page) or image floor plans
Set scale by clicking a known reference distance
Shade areas using polygon or rectangle tools
Group areas by zone with custom names and colors
Live area calculation in m² (2 decimal places)
Drag vertices to adjust shapes after placement
Trackpad-friendly: two-finger pan, pinch to zoom
Export printable summary with marked-up plan, area breakdown per group, project info, and timestamp

Use cases: AV system integration, interior fit-out, architecture, construction quantity takeoff, facility management, event space planning.
Getting started: Download area-takeoff-tool.html and open it in any modern browser. That's it.
